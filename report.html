The per-club WhatsApp ads

Alongside the main ad, we set up per-club lead reels that send an interested person straight into a WhatsApp chat with their own club, Yoga Reset and Afrobeats at Delta and TRM. They've only just started delivering, so the numbers are small, but early click-through is strong (4–11%).

One honest note: connecting the WhatsApp number so a click lands in the right club's chat was a bit of a challenge and took some sorting out. That's now done, so these per-club ads can run properly from here.

The takeaway

Paid is now a live channel, and the first creative is already beating the benchmark on clicks. Next: let the WhatsApp reels gather a proper week of data now that the number is connected, then put budget behind the winners.
